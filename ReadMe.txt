# 🌐 Signed Aonim – Personal Portfolio Website

A clean and minimal personal website built using **HTML**, **CSS**, and **JavaScript**.  
It showcases creative projects like *The Writer’s Game* and *Astra – Discord AI Bot*, and reflects a writing-inspired aesthetic.

## ✨ Features
- Responsive layout (works on desktop, tablet, and mobile)
- Smooth navigation and minimalist UI
- Links to GitHub projects and contact information
- Built for portfolio and freelancer showcase

### 🛠️ Technologies
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (for interactivity)
